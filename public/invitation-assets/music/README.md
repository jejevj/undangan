# Folder Musik Undangan

Simpan file musik (.mp3) di sini untuk digunakan sebagai background music undangan.

## Struktur
```
public/invitation-assets/music/
  perfect-ed-sheeran.mp3
  a-thousand-years.mp3
  cant-help-falling-in-love.mp3
  ...
```

## Aturan
- Format: MP3, ukuran maksimal 10MB per file
- File gratis: tersedia untuk semua user
- File premium: dikenakan biaya tambahan (belum diimplementasi)

## Cara pakai di undangan
Isi field "URL Lagu (mp3)" dengan URL lengkap, contoh:
`https://domain.com/invitation-assets/music/perfect-ed-sheeran.mp3`

Atau gunakan URL dari sumber eksternal (SoundCloud, dsb).
