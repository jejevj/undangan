# Copy Keys dari DOKU Dashboard

## Yang Perlu Anda Copy:

### 1. Secret Key
- Klik **"Reveal Key"** di sebelah "Active Secret Key"
- Copy secret key yang muncul
- Simpan sementara di notepad

### 2. DOKU Public Key
- Klik **"Salin"** di sebelah "DOKU Public Key"
- Paste ke notepad
- Simpan sebagai `doku_public_key_from_dashboard.txt`

### 3. Merchant Public Key
- Klik **"Salin"** di sebelah "Merchant Public Key"
- Paste ke notepad
- Simpan sebagai `merchant_public_key_from_dashboard.txt`

## Pertanyaan Penting:

**Apakah Anda masih punya Private Key yang match dengan Merchant Public Key yang sudah di-upload?**

### Jika YA (Anda masih punya private key):
1. Cari file private key Anda (biasanya `private.key` atau `merchant_private.key`)
2. Lanjut ke Step 2

### Jika TIDAK (Private key hilang):
Anda perlu generate ulang keys dan upload public key baru ke DOKU:
1. Generate keys baru
2. Upload public key baru ke DOKU (klik "Edit Merchant Public Key")
3. Lanjut ke Step 2
